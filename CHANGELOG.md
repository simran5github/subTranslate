# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-01-01

### Added
- Initial release of SubTranslate
- Real-time subtitle translation with MutationObserver
- Support for YouTube, Netflix, Disney+, Prime Video, Hulu, and Crunchyroll
- Multiple translation providers (LibreTranslate, Google Translate, MyMemory)
- Smart translation caching system
- Popup UI for settings management
- Language pair selection (12+ languages supported)
- Flicker reduction with smooth opacity transitions
- Cache statistics and management
- Chrome storage.sync for settings persistence
- MutationObserver-based subtitle detection
- Batch translation processing
- Per-platform CSS selectors for subtitle detection
- Error handling and logging
- Clean, modular ES6 code architecture

### Features
- ✨ Real-time subtitle translation
- 🚀 Smart caching to avoid duplicate API calls
- 🔧 Easy configuration via popup UI
- 💾 Persistent settings with chrome.storage.sync
- 📊 Cache statistics and monitoring
- 🌍 Multiple translation providers
- 📺 Support for major video platforms
- ⚡ Optimized performance with debouncing
